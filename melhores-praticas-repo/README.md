# Skill: Documentação Técnica de Modelos

Skill do Claude Code que gera automaticamente a documentação técnica de modelos
quantitativos (Excel, Python, VBA ou combinação) seguindo o template institucional
da área de risco de mercado.

## O problema que resolve

Documentar um modelo quantitativo manualmente é trabalhoso, inconsistente entre
autores e frequentemente deixado para depois. Esta skill automatiza esse processo
a partir do próprio artefato do modelo, garantindo documentação padronizada e
rastreável desde a entrega.

## Como começar

Leia `docs/guia-colaborador.md` — ele cobre o fluxo completo de trabalho em
co-criação, incluindo como usar o Claude Code para todas as operações do repositório.

## Estrutura

```
.claude/          código-fonte da skill
docs/             contexto do projeto e guia de colaboração  
tests/fixtures/   modelos sintéticos para teste
```
